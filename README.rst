===================
lib_dzne_basetables
===================

Backstory
---------

This project was designed to assist with the data tables of BASE_dzne. 

Installation
------------

To install lib_dzne_auto_interface, you can use `pip`. Open your terminal and run:

.. code-block:: bash

    pip install lib_dzne_basetables
